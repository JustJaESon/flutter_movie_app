
# Movie App
Movie app made in Flutter with clean architecture using BLoC, and the https://www.themoviedb.org/ API.

## Features
- Search feature
- Popular movies
- Top rated movies
- Movie details
- Movie cast
- Movie reviews
- Similar movies

## Installation
### 1. Clone the repo
```bash
git clone https://github.com/XcyzerZero/flutter_movie_app.git
cd flutter-movie-app
```
### 2. Run pub get
```bash
flutter pub get
```
### 3. Add your API key into api_constants file
```bash
apiKey = 'YOUR_API_KEY';
```
### 4. Run the app
```bash 
flutter run
```
